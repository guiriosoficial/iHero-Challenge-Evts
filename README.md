# iHero-Challenge-Evts

## Description
This is a base project for a challenge called iHero.
It`s emits websocket events randomicly to be handle by the main application.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# run server dev (watch mode)
$ npm run dev

# build production
$ npm run build

# start production
$ npm run start
```
